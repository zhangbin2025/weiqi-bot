# Weiqi Bot - 围棋 AI 助手

**对话式围棋助手** - 通过自然语言交互，完成棋谱下载、棋手查询、复盘分析、定式学习等任务。

👉 **在线体验**：https://bot.weiqi.lol/

## 核心功能

### 🎯 意图识别
- **自然语言理解**：识别用户意图，提取关键信息
- **智能跳转**：自动导航到对应功能页面
- **实体提取**：从对话中提取棋手名、棋谱链接、时间等参数
- **后台任务**：耗时操作后台执行，支持进度查询和取消
- **周期性任务**：定时执行重复任务（如每日查询）

### 📊 功能模块
| 功能 | 描述 |
|------|------|
| **棋谱下载** | 支持野狐、弈城等平台的棋谱下载 |
| **棋手查询** | 查询棋手信息、比赛战绩 |
| **对手分析** | 分析对手的棋风特点和对局数据 |
| **复盘分析** | AI 辅助复盘，标记关键局面 |
| **定式学习** | 定式词典、探索、发现、挑战 |
| **实战选点** | 从实战棋谱生成选点题 |
| **棋谱回放** | 支持分支变化的棋谱回放 |
| **记谱工具** | 辅助记录棋谱 |
| **赛事查询** | 查询围棋赛事信息 |

## 技术栈

- **前端**：TypeScript, Vite
- **存储**：IndexedDB, localStorage
- **平台**：Web (PWA) + Android (GeckoView) + Desktop (Electron)
- **架构**：分层架构（表现→应用→服务→基础设施→领域）
